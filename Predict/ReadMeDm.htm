Welcome to NeuralWorks Predict® and the world of neural computing! NeuralWorks Predict (we'll often call it simply Predict in this document) offers a complete and comprehensive solution for neural network data modeling.

 

This Predict Demonstration release allows you to explore many of Predict's capabilities before you purchase the software. The Demonstration release has the following restrictions:

 

·     The Demonstration release is only available for Windows/x86 architecture personal computers.

·     The Predict Command Line Interface is not provided.   Only the Microsoft Excel interface is available.

·     Models are limited to 32 fields and 512 records.

·     You cannot save any of the models that you build.

·     You cannot generate C, Visual Basic, or FORTRAN FlashCodeÔ source code.

·     You cannot write Variable Selection flags or the Data Analysis table to an Excel worksheet.

·     You cannot build Case Based Reasoning networks or perform Sensitivity and Explain analyses.

·     The Predict runtime Excel macro, PredictRun(), cannot be used.

·     The NeuralWorks Predict User Guide in Adobe® Acrobat® format is not provided on the Demonstration CD-ROM.  However, the CD-ROM does contain the complete Predict Windows Help file, which is linked to Predict to provide context-sensitive help, and the NeuralWorks Predict Getting Started Guide in Acrobat format.

 

** IMPORTANT RELEASE NOTES **

 

If the input data records for a model span multiple rows (or columns) in an Excel spreadsheet, and the spreadsheet also contains field names in multiple rows (or columns), when you specify the Excel range for Field Names on the Step 1 of 6 Dialog Box, you should ONLY specify the upper left cell of the range that contains field names.  If you select the entire range (or a partial range that spans multiple rows or columns), Predict will generate an error message.

 

Do not use parentheses or other punctuation characters in folder names.  The Predict Command-Line interface does not accept paths containing parentheses or other punctuation characters.  In particular, Windows XP64 Professional Edition users should avoid using C:\Program Files (x86) -> NeuralWare -> NeuralWorks -> Predict as your working directory, even though Predict is installed in that directory (the installation program recognizes the folder name).

 

Please note that due to bugs in various versions of Microsoft Excel, if display of the NeuralWare About NeuralWorks Predict form is enabled during Excel start-up, when you double-click an Excel spreadsheet from the Windows Explorer Excel will start but the spreadsheet will not open.  To resolve this problem, turn off the automatic display of About NeuralWorks Predict (click About NeuralWorks Predict in the Predict menu, and check the box labeled Do not show this form when Excel starts.).

 

In addition, currently Excel 2000 and Excel 2002 (XP) prevent using common Excel short cut keys (e.g., CTRL-Arrow) to select ranges when a Predict range input field is active. The latest service pack for Excel 2000 may fix the problem in Excel 2000; in any case, please install the latest Microsoft Office service packs whenever they become available.

 

What's New

Japanese and Simplified Chinese language versions of Predict are now available (Release 3.12).  Japanese and Simplified Chinese language versions of Predict will run on computers whose Locale setting matches the language that Predict was built for.  In all other Locales, the language is English.

 

Predict 3.x represents a major update of NeuralWorks Predict. In addition to a completely revised Graphical User Interface (GUI) for the Excel® version, Predict 3.x contains a full featured implementation of Kohonen Self-Organizing Maps to add more power to Predict's exploratory data analysis facilities.

 

Functions

Predict 3.x now supports building and running Self-Organizing Map (SOM) models.  Maps can have up to 4 dimensions, and up to 30000 processing elements in the Kohonen Layer.  Refer to Self-Organizing Maps in the Help file for more information.

 

The maximum number of data rows that Predict (in Excel) can use has been increased to 65,536 (the full size of an Excel spreadsheet). Command line files are limited only by available computing resources.  Demonstration release limitations increased slightly to 32 fields and 512 records.

 

When training ends for a Multi-Layer Perceptron (MLP) model, basic model performance is immediately evaluated and results displayed in a dialog box. MLP models are used for Prediction, Ranking, and Classification models.

 

The About NeuralWorks Predict dialog box now displays version information for the Operating System and Excel as well as Predict.

 

User Interface

The Predict GUI for Excel has been completely redesigned to take advantage of Visual Basic for Applications forms and controls, including Unicode fonts to support international character sets.

 

Model parameters are more logically organized on tabbed property pages as part of a new Model Parameters dialog box and new Preference property pages provide easy access to Predict operational parameters.  Default names for input and output fields can now be specified.

 

The status display that appears during model building has been revised; elapsed times for major steps are now displayed.

 

A number of improvements to the Command Line Interface have been made, including a new Command Line Interface that is more like UNIX and MS-DOS command lines.  The original Command Line Interface is also still provided.  Refer to the Command Line Interface Reference in Predict Help for more information (the Command Line Interface is not available in this Demonstration release).

Installing Predict Software

To ensure that Predict is automatically integrated with Excel, Excel must be installed before you install Predict.

 

All users MUST have Administrator privileges to install Predict.

 

To install Predict:

 

1.    Insert the NeuralWorks Predict CD-ROM into the drive.

2.    Wait for a few seconds for Windows to automatically run Setup.

        If Setup does not automatically start, manually start Setup:

·      select Run from the Windows Start menu.

·      type X:\SETUP in the Run dialog box (replace X with the appropriate CD-ROM drive letter).

3.    Follow the directions provided by Setup dialog boxes.

 

Installing Acrobat Reader

After Predict is installed, the Setup program may ask if you want to install Acrobat Reader 5.05.  If Acrobat Reader version 4.0 or later is not already installed on your computer, you should install the Acrobat Reader contained on the Predict CD-ROM so that you can view and print the NeuralWorks Predict Getting Started Guide for Windows.

 

Acrobat Reader installation is provided on CD-ROM distributions of Predict Demonstration, but it is not included in Email and Internet download distributions.  You can download Acrobat Reader www.adobe.com if you need a later version to view or print the Getting Started Guide.

 

If you have an older version of Acrobat Reader installed, but you still would like to install the newer version that NeuralWare provides, you MUST remove the older version of the Acrobat Reader before you start the Predict Setup program. If you do not remove the older version, you will not be able to install the newer version.

 

Also, note that the first time Acrobat Reader runs on your system, you will be asked to accept the Adobe License Agreement before the document you are trying to open is displayed. If you don't see the document you opened, make sure that another window isn't hiding the Acrobat Reader license acceptance dialog box.  After you accept the terms of the Adobe License Agreement, the document you opened will appear.

 

If you do not install Acrobat Reader during the Predict installation process, you can always install it from the Predict CD-ROM at a later time.  Browse to the acroread directory on the CD-ROM, and double-click the file named rp505enu.exe to install Acrobat Reader.

Removing the Predict Demonstration Product from your Computer

Removing the Predict Demonstration program is a two-step process.  If you are removing the Demonstration release in anticipation of installing the full Predict product, you do not need to perform the first step.

 

Step 1: Remove the Predict menu from Excel (not necessary if you plan to install the full product):

 

1.    Open Microsoft Excel if it is not already open.

2.   In the Microsoft Excel Tools menu, click Customize...

3.   With the Customize dialog box open, click the Predict menu in the Excel menu toolbar, and drag the menu off the toolbar.

4.   Click the Close button to close the Customize dialog box.

5.   Exit Excel.

 

Step 2: Remove Predict (button labels may differ from those shown below, depending on the Windows version in use):

 

1.   Open the Windows Control Panel.

2.    Open Add/Remove Programs.

3.    Select NeuralWorks Predict 3.2 Demo.

4.    Click Add/Remove.

5.    Click Yes to confirm that you want to remove the Predict Demonstration program.

6.   Click OK to close the Add/Remove Programs dialog box.
